# Comms Event Playbook — Private Investor Events

## Summary

A reusable workflow for Brooker private investor events, distilled from the two 2026
events the comms department ran: the [[2026-04-21-pantera-x-brooker-investor-lunch|Pantera
investor lunch]] and the [[2026-bnb-1k-gala-night|BNB $1K Gala Night]]. It is a working
reference for planning future events, not a formal policy.

## Event Format Pattern

| Element | Standard | Source basis |
|---------|----------|--------------|
| Audience | Family offices, institutional investors, key clients/partners, YPO members, founders | Both events |
| Size | ~40 capped for "meaningful dialogue" (lunch); larger mixed-stakeholder gala | Both events |
| Venue | 5-star Bangkok hotel near BTS; private function room, stage/podium, 2+ screens, 2–3 mics, registration table | Pantera brief |
| Structure | Arrival/networking → Brooker macro segment → guest/featured talk → Q&A → coffee networking | Pantera agenda |
| Branding | Co-branded with partner; backdrop, menu, signage | Pantera brief |

## Logistics Checklist

1. Confirm date (+ alternative) and check venue availability.
2. Shortlist 5-star hotels near BTS — deliver 3 options with pricing + budget estimate
   within ~3 days.
3. Book venue; arrange AV setup and lunch menu (appetizer, main, dessert, coffee).
4. Prepare invitation list; send invitations targeting RSVP confirmation ~4 weeks out.
5. Track RSVPs; send a 2-week and a 3-day reminder.
6. Coordinate speakers (Brooker + guest partner).
7. Produce event signage, backdrop, menu, and registration list.

## Speaker & Content Workflow

- Brooker delivers an opening macro segment — see
  [[macro-outlook-geopolitical-recalibration]].
- The guest partner delivers the featured talk.
- The CEO keynote is drafted in multiple tone variants and rehearsed — see
  [[the-audacity-of-value-speech]] for an example of the drafting approach.
