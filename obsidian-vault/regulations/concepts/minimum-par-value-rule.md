# Minimum Par Value for Listed Shares — SET Circular บจ.(ว) 14/2559

## Summary

SET circular **บจ.(ว) 14/2559** (dated 23 September 2559 / 2016) addresses the
**minimum par value** (มูลค่าที่ตราไว้ขั้นต่ำ — "PAR") for shares of companies
applying for, or maintaining, listed status on the Stock Exchange of Thailand.

## Key Point

- Listed ordinary shares must have a **par value of not less than 0.50 baht per
  share**, fully paid. This minimum-PAR floor is one of the **share-quality
  qualifications** carried forward into the [[set-ipo-listing-criteria-2025]] admission
  criteria.

## Archive Note

The source PDF is an image-only scan; the 0.50-baht floor is corroborated by the SET
IPO admission criteria (`IPO-1` document), which states common shares must have par
value ≥ 0.50 baht/share. For any par-value reduction (split) actions, consult current
SET rules.
