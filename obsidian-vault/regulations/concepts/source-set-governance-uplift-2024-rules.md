# Source: SET Governance Uplift — Rule Editions 1–14 (2024)

## Document Overview

The detailed SET rule package enclosed with circular บจ. (ว) 2/2567 — see
[[listed-company-governance-uplift-2024]] for the substantive analysis. Source files
modified 21 February 2024.

## Key Findings

- **ข้อกำหนดของตลท. ฉบับที่ 1–7** — the first seven revised SET regulation editions
  covering listing admission and the supervision of at-risk companies.
- **ข้อกำหนดของตลท. ฉบับที่ 8–14** — editions 8–14, continuing the revised
  disclosure, warning-mark, and off-criteria-company provisions.
- **แนวทางดำเนินการ** — SET's implementation guidance/approach explaining how the
  revised rules are applied in practice.

## Relevance

These are the operative SET texts behind the 2024 governance uplift. They are consumed
through the parent concept; individual editions are procedural amendments without
free-standing thesis content.
