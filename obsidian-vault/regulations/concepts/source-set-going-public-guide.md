# Source — SET Going Public Guide

## Document Overview

- **As-of date:** booklet, file modified 26 January 2015.
- **Issuer:** The Stock Exchange of Thailand, New Listing Department.
- **Folder:** `SET Publishing_Listing`. Thai and English editions exist.

## Key Findings

- The booklet explains the listing criteria and the listing process applicable to both
  SET and mai.
- Provides the institutional contact directory for the listing ecosystem (SET, SEC,
  FETCO, TLCA, maiA, ASCO, AIMC, FAP).

## Relevance

Primary practitioner guide behind [[going-public-listing-process]]. Note that the
numerical admission thresholds it cites predate the 2025 upgrade — see
[[set-ipo-listing-criteria-2025]].

## Extracted Data

The folder also bundles the SET Listing Guide, SET Manual 2014, an IPO-preparation
manual, a Spin-Off brochure and the IOD Handbook 2015 — catalogued in the concept
article.
