# Going Public — SET Listing Process & Guides

## Summary

The "SET Publishing / Listing" materials are the Stock Exchange of Thailand's
**practitioner guides on the process of going public** — how a company prepares for
and executes a listing on the main board (SET) or the Market for Alternative
Investment (mai). They explain listing criteria and the step-by-step listing process
for both boards. The project archive copies date from circa **2014–2016**.

## Key Points

- The **Going Public Guide** (Thai and English editions) explains the listing criteria
  and process applicable to both SET and mai.
- The **SET Listing Guide / SET Manual 2014** and `เตรียมตัวอย่างไรได้เข้าตลาด...`
  cover the preparation steps a company should take before applying.
- A dedicated **Spin-Off brochure** covers the listing of a subsidiary spun off from
  an existing listed parent.
- The **IOD Handbook 2015** provides director-level guidance relevant to companies
  preparing for listed status.
- Key institutional contacts named in the guides: SET New Listing Department, the Thai
  SEC, FETCO, TLCA, maiA, ASCO, AIMC and FAP.

## Historical Context

These guides describe the *process* of listing; the *numerical admission thresholds*
were subsequently raised — see [[set-ipo-listing-criteria-2025]] for the criteria
effective 1 January 2025. Use the guides for process orientation and the 2025 concept
for current thresholds.
