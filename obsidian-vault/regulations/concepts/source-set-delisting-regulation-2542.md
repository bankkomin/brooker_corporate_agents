# Source: SET Delisting Regulation B.E. 2542

## Document Overview

**As-of:** the **Stock Exchange of Thailand regulation on the delisting of listed
securities, B.E. 2542** (ข้อบังคับตลาดหลักทรัพย์ฯ ว่าด้วยการเพิกถอนหลักทรัพย์จดทะเบียน
พ.ศ. 2542), in the version **effective 1 February 2561 (2018)**. Two copies are held in
the `Collection of Laws` folder of the Thai SEC regulations archive.

## Key Findings

- This SET regulation is the **formal rulebook** behind the delisting process: it sets out
  the grounds on which SET may delist a security, the procedure SET follows, and the
  conditions for a company to have the delisting grounds removed.
- It is the authoritative source underlying the plain-language [[delisting-criteria]]
  concept, which is built from a SET simplified-regulations summary.
- The B.E. 2542 (1999) regulation has been amended; the copy here is the version effective
  1 February 2018.

## Relevance

For a listed company, this is the binding regulation governing the worst-case outcome —
loss of listed status. It pairs with the SP/NC marking procedure and the financial,
audit-opinion, and disclosure thresholds that define delisting risk.
