# Source — SEC Guidance on Pre-Filing Consultation for Share-Offering Applications

## Document Overview

SEC guidance "Approach to Consulting the SEC Office Before Filing a Share-Offering
Application" (แนวทางการเข้าหารือกับสำนักงาน ก.ล.ต. ก่อนการยื่นคำขออนุญาตเสนอขายหุ้น) —
revised edition, circulated January 2559 (2016); source file modified 9 March 2016. It
was drafted jointly by the SEC and the Investment Banking Club.

## Key Findings

The SEC encourages a **financial advisor (FA)** to consult the SEC **before** filing a
share-offering application, so that key qualification and disclosure issues are surfaced
and resolved early. Benefits cited: a cleaner application, and faster processing within
the timeframes of the **Licensing Facilitation Act B.E. 2558**.

For the pre-filing consultation an FA should prepare, at minimum, material covering:

1. **Key risk areas** of the issuer.
2. The FA's **due diligence** on those key risk areas, and the FA's observations from it.
3. Issues raised in the FA's discussions with the **auditor and internal auditor**, and
   how the FA addressed them.
4. Areas the FA still considers **risk areas** requiring SEC discussion before filing.
5. For a **re-filed case** (a prior application that was withdrawn or rejected): if the
   same FA, a summary of the reasons for the prior withdrawal/rejection and how each issue
   was fully resolved; if a new FA, an equivalent review.

## Relevance

Practical procedural guidance for any Brooker IPO/listing transaction — the FA should use
the pre-filing consultation channel to de-risk the application. Supports the
[[sec-financial-advisor-and-listing-guidance]] concept.
