# Record Date for Granting Shareholder Rights — SET Circular บจ.(ว) 6/2560

## Summary

SET circular **บจ.(ว) 6/2560** concerns the **revision of the Record Date** mechanism
used to determine which shareholders are entitled to a given right (dividends, rights
issues, meeting attendance, etc.). The **Record Date** is the cut-off date on which a
listed company fixes the list of shareholders eligible for a corporate right. The
circular was distributed in 2017 (the project archive holds the SET circular plus a
24 August 2017 forwarding email).

## Key Points

- The Record Date replaced the older "book-closing date" approach as the standard
  method for SET-listed companies to identify shareholders entitled to a right.
- The circular updated the Record Date practice so the eligible-shareholder list is
  fixed on a defined date for any right granted to shareholders.
