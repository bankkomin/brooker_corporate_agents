# Source Summary — ARK On-Chain Data: A Framework to Evaluate Bitcoin

_As of 30 December 2021. Authors: Yassine Elmandjra & David Puell, [[ark-invest]] (data from [[glassnode]])._

## Document Overview

An ARK white paper presenting a structured framework for fundamentally analyzing
Bitcoin using public on-chain data — addressing the objection that Bitcoin "cannot be
analyzed fundamentally."

## Key Findings

- Bitcoin analysis organized as a **three-layer pyramid**:
  - **Layer 1 — Network Health**: monetary integrity, security (hash rate), usage
    (active addresses, transaction volume).
  - **Layer 2 — Buyer & Seller Behaviour**: UTXOs, coindays destroyed, on-chain
    profit/loss, realized cap, thermo cap, HODL Waves.
  - **Layer 3 — Valuation**: MVRV ratio, MV-to-thermo-value, investor capitalization,
    short/long-term realized value, profit/loss oscillators.
- On-chain metrics give Bitcoin a fundamental toolkit comparable to EV/EBITDA for
  equities.

## Relevance

The foundational document for the [[on-chain-analysis-framework]] concept. Precursor to
the 2023 ARK x Glassnode Cointime Economics work ([[source-ark-glassnode-cointime-economics-2023]]).
