# Token Value Accrual Mechanisms

## Summary

A tokenomics concept drawn from Delphi's "Crypto Insiders" deep dives: a token's price is
only sustainable if the protocol routes real economic value to it. "Governance" alone is
a weak value-accrual mechanism; the research argues for cash-flow-like designs.

## Mechanisms Discussed

- **Fee capture / buyback-and-burn** — protocol revenue used to buy and burn the token,
  making it deflationary (BNB's burn program is a live example — ~22% of supply burned
  by Q1 2023).
- **Staking / real yield** — fees distributed to stakers rather than emitted as inflation.
- **Collateral / backing demand** — the token is required as collateral (Synthetix:
  SNX backs synthetic assets; the Cosmos ATOM 2.0 proposal would use ATOM as interchain
  security collateral).
- **Governance** — weakest on its own; valuable only when bundled with cash-flow rights.

## Why It Matters

The 2022 bear market punished tokens whose only "use" was governance. Reports stress
distinguishing **fundamental value accrual** from **market-sentiment-driven** value — e.g.
Messari's BNB analysis showed network value rising while revenue (in BNB) fell, implying
sentiment, not utility, drove price. Blockchain assets may need bespoke valuation models
(e.g. an "Expected Demand for Security" model) rather than simple P/S ratios.
