# Source: It's All About Debasement — Macro Insiders In Focus (Aug 2023)

## Document Overview

As-of date: **August 2023** (extract `_Modified:` 2024-09-10; the file is named
`MI_In_Focus_AUG22` but page footers read "August 2023"). Author: Raoul Pal, Macro
Insiders In Focus ([[global-macro-investor]] / Real Vision) — a section of GMI shared
to add context to The Everything Code.

## Key Findings

- "It's all about debasement" — total liquidity (GMI's debasement indicator) is the key
  driver of [[the-everything-code]].
- The GMI Weekly Liquidity Index (WLI) year-on-year rate of change was rising but still
  below zero; debasement "starting to increase again."
- The Everything Code logic — interest-payment monetisation — implies the Fed balance
  sheet begins to grow in Q4 (Pal attributes the trigger to the banks).
- The Global WLI hides divergences beneath the surface (e.g. the BoJ balance sheet in
  yen terms).

## Relevance

debasement to the [[global-liquidity-cycle]].
