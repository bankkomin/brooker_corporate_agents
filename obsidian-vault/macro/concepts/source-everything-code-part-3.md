# Source: The Everything Code Part 3 — Macro Insiders Deep Dive (Jun 2023)

## Document Overview

As-of date: **27 June 2023** (extract `_Modified:` 2024-09-10). Author: Raoul Pal,
Macro Insiders Deep Dive ([[global-macro-investor]] / Real Vision). **Part 3** — the
synthesis, presented ahead of an Everything Code AMA session.

## Key Findings

- Brings the framework together. The established premises:
  - Government debt growth is almost entirely driven by interest payments.
  - Those payments end up on the Fed's balance sheet.
  - Financial crises add directly to the balance sheet on top of monetised interest.
  - All central banks are doing the same and probably working together.
- The Fed will likely add more than just interest-payment monetisation to re-liquefy
  the banks — projecting a Fed balance sheet of ~US$12-14tn by end-2025.
