# Eastbound (Singapore Placement Agent)

## Overview

Eastbound is a Singapore-based placement agent engaged by Brooker Group to raise external
LP capital for the VCC fund platform. It was ratified as a primary distribution partner in
[[2026-02-21-strategic-partner-map]].

| Attribute | Value |
|-----------|-------|
| Type | Placement agent |
| Location | Singapore |
| Client base | ~500 clients across Asia and Europe |
| Mandate | Help raise toward the global $60M VCC fundraising goal (part of the USD 600M AUM target) |
